0x01 Function

接受一个参数，返回一个参数，是处理一个对象，所以是功能

0x03 Supplier

不用接受任何输入参数，获得一个固定的结果，所以他是提供者

一般在单例模式里面可以用这个，但是没想不出来模拟使用场景，所以没有例子。

注意，无输入参数！

0x04 UnaryOperator

这个，还没看出怎么用，他继承了Function接口，所以功能跟Function接口类似

0x05 BiFunction

接受两个参数，处理之后返回一个值
